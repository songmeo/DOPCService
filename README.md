# Delivery Order Price Calculator service (DOPC)

## Quick Start
Install Nox:

```commandline
pip install nox
```

Run nox to install dependencies and run tests:
```commandline
nox
```

Run test server on `localhost:8000`:
```commandline
nox -s server
```
_That's it! :3_

Please check `noxfile.py` for more available commands.
